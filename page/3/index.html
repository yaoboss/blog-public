ifconfig command not found

找不到ifconfig命令

新安装的centos7，准备配置网络设置时，使用ifconfig命令，发现居然找不到命令，command not found

问了下谷老师，看来linux也是更新了不少啊

ifconfig命令属于net-tools，但是centos7的默认minimal最小化安装中并没有安装net-tools

而且net-tools现在属于被废弃的，官方已经不推荐使用
所以有两种解决办法：
1.当然就是直接安装net-tools, yum install net-tools

2.使用 ip a，命令代替，官方也推荐这种做法

git warning: LF will be replaced by CRLF in xxxx解决办法

发表于 2016-05-01   |   分类于 git   |  

这两天在家改用git的命令行bash工具了，在公司一直都是用smartgit，心血来潮也玩下bash.

不过在使用的时候，好多警告真的是好烦，比如：

在 git add xx.file 的时候，提示

warning: LF will be replaced by CRLF in XXXXXXXXXXXXXX.

大概好像是换行符转换的问题，网上搜了下可以直接关闭这种警告，这样干：

it config core.autocrlf false

然后就不会在有之前的那个警告啦。
